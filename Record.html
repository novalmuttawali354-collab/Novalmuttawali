Screen Video Recorder

Rekam layar komputer Anda langsung dari browser Chrome

Pratinjau rekaman akan muncul di sini

Status: Siap merekam
00:00:00

Cara Menggunakan:

  • Klik tombol "Mulai Rekaman" untuk memulai merekam layar Anda
  • Browser akan meminta izin untuk merekam layar - pilih layar atau jendela yang ingin direkam
  • Gunakan tombol "Hentikan Rekaman" untuk menghentikan perekaman
  • Setelah selesai, gunakan tombol "Unduh Video" untuk menyimpan rekaman ke komputer Anda
  • Rekaman akan disimpan dalam format MP4 yang kompatibel dengan kebanyakan pemutar video

Screen Video Recorder untuk Chrome | Dibuat dengan HTML5, MediaRecorder API, dan JavaScript

Catatan: Aplikasi ini hanya berfungsi di browser modern yang mendukung MediaRecorder API (Chrome, Edge, Opera)
